- .bio 👉 To Change Userbot Bio

- .join 👉 To Join Channel Or Group Link/Username

- .pjoin 👉 To Join Private Channel

- .leave 👉 To Leave The Chat

- .spam 👉 Spam In Group Or Id Only 100 Time

- .dreplyraid 👉 ( Deactivation )

- .raid 👉 ( Abused Raid )

- .delayspam 👉 Delay
